## 一、项目概述

本项目是一个基于 **纯 HTML + D3.js v7** 的单页数据可视化叙事网站，以"报纸风格"展示 **1949-2025 年中国粮食安全** 的完整图景。

### 核心交互模式

| 交互 | 实现方式 |
|------|---------|
| **主页面浏览** | 纵向滚动，图表随滚动依次以动画入场 |
| **深度分析** | 点击"查看详情"，覆层从右侧滑入（全屏） |
| **返回主页** | 点击"← 返回" / 右上角 ✕ / 按 Esc 键 / 点击遮罩 |
| **深度页内切换视图** | 控件按钮实时切换图表类型/数据维度 |

### 技术栈

- **D3.js v7**（CDN）：所有图表渲染
- **纯 CSS 动画**：滚动入场 + 覆层滑入/滑出
- **IntersectionObserver**：滚动触发懒渲染
- **零依赖**：无框架，无构建工具，纯静态文件

## 三、网页结构

### 3.1 主页面（仪表盘）

所有主内容在 `<div class="container">` 中，**始终可见**，不再通过 `display:none` 切换。

```
body
├── div.container
│   ├── header.masthead          ← 报纸报头（加载即动画入场）
│   ├── section#page-dashboard
│   │   ├── h2.section-title     ← "关键指标一览"
│   │   ├── div#metric-cards     ← 5个指标卡片（reveal-section）
│   │   ├── div.divider          ← 分隔线（reveal-section）
│   │   ├── div.chart-grid × 5   ← 4行图表（均为reveal-section）
│   │   │   ├── Row1: [人均粮食占有量] [分品种堆叠面积]
│   │   │   ├── Row2: [消费结构流图（全宽）]
│   │   │   ├── Row3: [全球对比柱状] [粮食安全雷达]
│   │   │   └── Row4: [关键历史节点时间线（全宽）]
│   └── footer
│
├── section#page-deep-1 ~ 8      ← 深度页覆层面板（position:fixed, 初始在屏幕右侧外）
├── div.deep-overlay-backdrop    ← 覆层背景遮罩（JS动态创建）
└── button.deep-overlay-close    ← 覆层关闭按钮（JS动态创建）
```

### 3.2 滚动入场机制

每个需动画入场的区块添加 `.reveal-section` 类：

```css
.reveal-section {
  opacity: 0;
  transform: translateY(50px);
  transition: opacity 0.9s, transform 0.9s;
}
.reveal-section.visible {
  opacity: 1;
  transform: translateY(0);
}
```

- **IntersectionObserver** 以 `threshold: 0.12, rootMargin: -60px` 监听所有 `.reveal-section`
- 区块进入视口时添加 `.visible` 类，触发 CSS 过渡
- 指标卡片额外使用 `.stagger-children` 实现子元素依次淡入
- 图表渲染也由同一观察器触发（懒渲染），渲染完成后 SVG 自带 scale 入场动画

## 四、覆层系统（替代旧版 hash 路由）

### 4.1 架构

```
Overlay 对象（router.js）
├── init()          创建背景遮罩+关闭按钮，绑定Esc键
├── open(route)     打开指定深度页覆层
├── close()         关闭当前覆层
└── _hideCurrent()  内部：移除 overlay-active 类
```

### 4.2 打开流程

```
用户点击 "查看详情 →" 或 D3 图表区域
    │
    ├── Overlay.open('deep/N')
    │   ├── 获取对应 section#page-deep-N
    │   ├── 添加 .overlay-active 类 → CSS transition 滑入
    │   ├── document.body overflow → hidden（禁止背景滚动）
    │   ├── 显示背景遮罩（淡入）
    │   ├── 300ms 后显示关闭按钮
    │   └── 触发 overlay-open 自定义事件
    │       └── DeepPages.render('deep/N')
    │           ├── 首次：DataLoader.loadAll()
    │           └── 渲染对应深度页图表
```

### 4.3 关闭流程（4种触发方式）

| 方式 | 触发 |
|------|------|
| 点击"← 返回仪表盘" | `Overlay.close()` |
| 点击右上角 ✕ 按钮 | `Overlay.close()` |
| 点击深色背景遮罩 | `Overlay.close()` |
| 按键盘 Esc | `Overlay.close()` |

关闭时：
1. 移除 `.overlay-active` → CSS transition 滑出
2. 遮罩淡出，关闭按钮隐藏
3. 恢复 `body` 滚动
4. 触发 `overlay-close` 事件 → `DeepPages.cleanup()`（停止播放动画等）

### 4.4 CSS 动画细节

```css
.deep-page {
  position: fixed;
  transform: translateX(105%);    /* 默认隐藏在右侧屏幕外 */
  transition: transform 0.55s cubic-bezier(0.22, 1, 0.36, 1);
  visibility: hidden;
}
.deep-page.overlay-active {
  transform: translateX(0);       /* 滑入到正常位置 */
  visibility: visible;
}
```

- 使用 `cubic-bezier(0.22, 1, 0.36, 1)` 缓动函数，提供自然的"减速停车"感觉
- `visibility: hidden` 确保隐藏时不可交互且不影响键盘焦点

## 八、数据预处理

`process_real_data.py` 从原始CSV生成标准化数据文件：

- **grain_output.csv** / **per_capita_grain.csv**：基于国家统计局**真实数据**（人均=总产/人口）
- **diet_structure.csv** / **self_sufficiency.csv** / **agri_technology.csv** / **arable_land.csv**：基于已知趋势的**线性插值估算**
- **world_comparison.csv**：2023年截面估算数据
- **key_events.json**：手动维护的16个关键历史节点
- **fao_production_value.csv** / **fao_top_crops.csv**：FAO单品数据按7大品类聚合

## 十、快速启动

纯静态文件，无需构建工具。用任意 HTTP 服务器：

```bash
cd 吃得饱
python -m http.server 8000
# 或: npx serve .
```

浏览器打开 `http://localhost:8000`。
