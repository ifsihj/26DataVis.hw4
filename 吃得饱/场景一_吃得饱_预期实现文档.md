# 场景一：吃得饱 — 数据可视化预期实现文档

> 项目定位：数据可视化课程团队项目 · 场景一
> 叙事主题：中国人如何从"吃不饱"到"吃得饱"
> 技术栈：D3.js (v7) + 原生 JavaScript SPA
> 视觉风格：新闻叙事风（报纸质感数据新闻）

---

## 一、项目概述

### 1.1 核心叙事线

"中国人如何解决吃饱的问题"——这条主线分为三个递进层次：

1. **总量够不够**（粮食产量与人均占有量的历史跨越）
2. **饭碗端没端牢**（自给率、耕地红线、库存安全）
3. **凭什么增产**（科技进步对产量的贡献）

整体以 **横向（时间线）** 为主轴，纵向（国际对比、各维度对比）为辅，不跳出"吃得饱"这个主题边界。

### 1.2 三场景关系

本项目为三场景之一（吃得饱 / 吃得好 / 吃得负责）。本文档仅覆盖"吃得饱"部分，设计时留有接口（统一的色板、交互规范），便于后续与其他场景整合。

## 二、数据假定说明

> 以下数据为**基于公开发表趋势的合理假定**，用于原型开发。后续可替换为真实数据。
> 数据量级参考来源：国家统计局《中国统计年鉴》、FAO、世界银行 WDI、农业农村部公报。

### 2.1 假定数据集总览

| 数据集 | 时间范围 | 粒度 | 关键字段 |
|--------|---------|------|---------|
| `grain_output.csv` | 1978-2025 | 年度 | 年份、总产量(万吨)、稻谷、小麦、玉米、豆类、薯类 |
| `per_capita_grain.csv` | 1978-2025 | 年度 | 年份、人均占有量(kg/年)、FAO安全线(400kg) |
| `diet_structure.csv` | 1978-2025 | 年度 | 年份、谷物直接消费%、肉禽%、蛋奶%、蔬菜%、其他% |
| `self_sufficiency.csv` | 2000-2025 | 年度 | 年份、稻谷自给率、小麦自给率、玉米自给率、大豆自给率 |
| `agri_technology.csv` | 1985-2025 | 年度 | 年份、亩产(kg)、农业专利授权数、增产相关论文数、杂交稻覆盖率% |
| `world_comparison.csv` | 2023 (截面) | 国家 | 国家、人均粮食占有量、自给率、亩产、人均耕地 |
| `arable_land.csv` | 1978-2025 | 年度 | 年份、耕地面积(亿亩)、人均耕地(亩)、粮食播种面积(亿亩) |

### 2.2 各数据集假定逻辑

> **⚠️ 重要说明**：以下每个数据集的趋势方向、量级范围都基于公开统计数据，但**具体数值为教学演示目的而构造**。替换真实数据时调整 CSV 文件即可，D3 可视化代码无需改动。

#### `grain_output.csv` 粮食总产量

假定逻辑：从 1978 年约 3.0 亿吨持续增长，2000 年突破 4.6 亿吨，2010 年突破 5.4 亿吨，2023 年约 6.9 亿吨。中间有少数年份因自然灾害小幅回调（如 2003 年）。

#### `per_capita_grain.csv` 人均粮食占有量

假定逻辑：1978 年约 300kg/人，低于 FAO 400kg 安全线。1990 年代逐步接近 400kg，2010 年后稳定在 440-480kg。国际对比：世界平均约 350kg，美国约 1200kg，印度约 230kg。

#### `diet_structure.csv` 主食消费结构

假定逻辑：1978 年谷物直接消费占食物总消费约 65%（"吃饱"阶段），此后逐年下降，2025 年约 30%。肉禽、蛋奶占比从约 10% 上升至约 35%。这反映了从"吃饱"到"吃好"的过渡——虽然是"吃得饱"场景，但消费结构变迁是"吃饱"问题解决后的自然结果，作为辅助叙事。

#### `self_sufficiency.csv` 粮食自给率

假定逻辑：稻谷、小麦始终维持 95%+ 自给。玉米在 2010 年后因饲料需求上升，自给率从 99% 降至约 90%。大豆自给率从 2000 年约 50% 持续降至约 15%（此为粮食安全的核心挑战之一）。

#### `agri_technology.csv` 科技指标

假定逻辑：亩产从 1978 年约 180kg 提升至 2023 年约 390kg。农业专利授权数从 1985 年（专利法实施）的极低基数快速增长，2010 年后出现爆发式增长。杂交稻覆盖率从 1978 年约 10% 上升至 2020 年约 55%。

#### `world_comparison.csv` 全球对比

假定逻辑：选取中国、印度、美国、巴西、日本、尼日利亚、世界平均等 7 个实体。中国的人均粮食占有量处于中等偏上，亩产领先，人均耕地远低于世界平均。

#### `arable_land.csv` 耕地数据

假定逻辑：耕地总面积从 1978 年约 20 亿亩（约 1.35 亿公顷）逐渐下降，2010 年后在 18 亿亩红线附近保持稳定。人均耕地持续下降（人口增长 + 城市化）。

## 三、可视化架构

### 3.1 信息架构

```
[仪表盘总览页]  ← 默认视图
  ├── 顶部：关键数字卡片（当前人均占有量 / 总产量 / 自给率 / 亩产）
  ├── 行1：人均粮食占有量时间线（折线图） + 粮食总产量变化（面积图）
  ├── 行2：主食消费结构变迁（堆叠面积图 / 百分比） 
  ├── 行3：全球对比（分组柱状图） + 粮食安全雷达图
  │
  └── 每个图表均可点击 → 进入对应的深度页

[深度页 1] 人均粮食占有量详解
  ├── 长篇时间线（带关键事件标注：1978 改革开放、2000 年、2020 全面小康）
  ├── FAO 安全线标注
  └── 国家对比选择器

[深度页 2] 粮食总产量详解
  ├── 堆叠面积图 + 各粮食品种切换
  ├── 历年增产/减产标注（政策、灾害）
  └── 时间范围刷选器

[深度页 3] 消费结构详解
  ├── 百分比堆叠面积（主图）
  ├── 各品类绝对量折线（折线面板）
  └── 视图切换：百分比 / 绝对量

[深度页 4] 科技驱动增产详解
  ├── 亩产增长曲线 + 关键科技节点标注
  ├── 专利 & 论文数量双轴图
  └── 杂交稻覆盖率面积图

[深度页 5] 粮食安全综合指标
  ├── 各品种自给率排名（棒棒糖图）
  ├── 耕地红线面积表（面积图）
  └── 世界主要国家自给率对比
```

### 4.1 布局

```
┌────────────────────────────────────────────────────┐

│  副标题 / 时间范围 / 数据来源标注                │
├───────┬───────┬───────┬───────┬───────┤

├───────┴──┬────┴───┬───┴───┬───┴───┤

│  占有量  │ 产量   │ 费结构 │ 对比  │

│          │        │       │       │

```

### 4.2 顶部数字卡片

- 5 个关键指标，显示最新年份（2025）数据
- 带小号 Δ 变化指示（相比上一年或相比 1978 年）
- 点击数字可短暂高亮对应图表

### 4.3 各图表区域

每个图表包含：
- 左上角：图表标题（如"人均粮食占有量"）
- 右上角：点击入口 → 深度页的提示图标（🔍 或 "查看详情 →"）
- 右下角：小字号数据来源标注
- 基础交互：tooltip、图例点击切换

### 4.4 仪表盘交互行为

- **点击图表** → 切换到对应深度页（带动画过渡）
- **hover tooltip** → 显示精确数值
- **图例点击** → 显示/隐藏序列
- **鼠标滚轮** 在图表区域 → 不触发页面滚动，用于缩放（深度页）

### 5.2 深度页 1：人均粮食占有量详解

**核心图表**：长格式折线图（1978-2025），横向可滚动
**关键标注**：
- 1978：改革开放，农业生产队改革
- 1982：家庭联产承包责任制全面推行
- 1996：粮食总产突破 5 亿吨
- 2003：粮食产量低谷（自然灾害+结构调整）
- 2013：人均粮食占有量首次突破 450kg
- 2020：全面小康，人均粮食占有量 > 470kg
**交互**：
- 时间范围刷选器（brush）
- 叠加 FAO 安全线（400kg）切换
- 添加其他国家对比曲线（下拉选择）

### 5.3 深度页 2：粮食总产量详解

**核心图表**：堆叠面积图（稻谷、小麦、玉米、豆类、薯类）
**辅助图表**：增长率柱状图（同比上年）
**注释**：
- 1990 年代玉米占比快速上升（饲料需求）
- 2003 年减产事件标注
- 各品种占比变迁
**交互**：
- 单独显示/隐藏某个品种
- 切换至百分比视图
- 点击某品类跳转到该品类历年条形图

### 5.4 深度页 3：主食消费结构详解

**核心图表**：百分比堆叠面积图（谷物直接消费、肉禽、蛋奶、水产、蔬菜、其他）
**辅助图表**：人均肉类消费量折线（kg/年）
**叙事重点**：从"吃饱"（谷物为主）到"吃好"（肉蛋奶占比上升），食品消费升级是"吃饱"问题解决后的直接结果
**交互**：
- 百分比 ↔ 绝对量视图切换
- 品类选择器（显示/隐藏特定品类）
- 年份滑动播放（自动播放动画）

### 5.5 深度页 4：科技驱动增产详解

**核心图表组合**：
1. 亩产折线图（左轴） + 专利/论文柱状图（右轴）——双轴图
2. 杂交稻推广面积覆盖率面积图（底部）
**关键标注**：
- 1973：袁隆平团队实现杂交稻三系配套
- 1985：中国专利法实施
- 2000：超级稻亩产 700kg 目标达成
- 2014：超级稻亩产突破 1000kg
**交互**：
- 三类指标（亩产、专利、论文）的显示切换
- 点击标注点弹出详细说明

### 5.6 深度页 5：粮食安全综合指标

**核心图表**：
1. 各粮食品种自给率棒棒糖图（lollipop chart）
2. 耕地面积变化面积图 + 18 亿亩红线标注
3. 主要国家自给率对比（分组柱状图）
**叙事重点**：
- "口粮绝对安全"（稻谷、小麦近 100% 自给）

- 耕地红线的坚守与挑战
**交互**：
- 点击品种查看进口来源国分布
- 时间滑动查看耕地变化

## 六、交互设计规范

### 6.1 通用交互

| 交互类型 | 触发方式 | 响应 |
|---------|---------|------|
| Tooltip | hover 数据点 | 精确数值 + 单位 |

| 返回 | "← 返回"按钮 / 浏览器返回 | 右滑动画返回仪表盘 |

### 6.2 深度页专用交互

| 交互类型 | 用途 | 实现方式 |
|---------|------|---------|
| Brush 刷选 | 时间范围选择 | d3-brush |

| 视图切换 | 百分比/绝对量 | 按钮组 toggle |

| 缩放 | 细节查看 | d3-zoom |
| 点击钻取 | 查看子类别详情 | 切换到子视图 |

### 6.3 响应式原则

- 仪表盘优先适配 1440×900 及以上分辨率
- 图表在 1024px 以下进行布局压缩（图例下移、减小编号）
- 深度页核心图表区域最小 600px 高

## 八、技术方案

### 8.1 为什么选 SPA

| 维度 | SPA（单页） | 多 HTML 文件 |
|------|-----------|------------|
| D3 状态共享 | 全局变量共享，联动自然 | 需要 localStorage/URL 传参 |
| 动画过渡 | 页面切换动画流畅 | 页面刷新，无过渡 |
| 数据加载 | 启动时一次性加载 | 每页重复加载 |
| 部署 | 一个目录搞定 | 多文件管理 |
| 学习成本 | 略高（需简单路由） | 零门槛 |

**结论**：本项目选择 SPA + Hash 路由，平衡开发体验与效果。

### 8.2 路由设计

```
#dashboard        → 仪表盘（默认）
#deep/1           → 人均粮食占有量详解
#deep/2           → 粮食总产量详解
#deep/3           → 主食消费结构详解
#deep/4           → 科技驱动增产详解
#deep/5           → 粮食安全综合指标
```

路由变化时：隐藏当前 section → 过渡动画 → 显示目标 section → 调用对应图表的 render 函数。

### 8.3 关键 D3 模块

| 图表类型 | D3 模块 | 使用场景 |
|---------|---------|---------|
| 折线图 | d3-line, d3-axis | 人均占有量、亩产 |
| 面积图 | d3-area | 总产量、耕地变化 |
| 堆叠面积图 | d3-stack, d3-area | 消费结构、分品种产量 |
| 分组柱状图 | d3-band | 全球对比 |
| 雷达图 | d3-line (极坐标) | 安全指标 |
| 棒棒糖图 | d3-bubble, d3-line | 自给率排名 |

## 十、实施风险与备注

1. **数据替换**：所有数据通过 CSV 文件加载，图表代码与数据解耦。替换真实数据时，只需修改 CSV，无需改动 JS。
2. **国际化**：如需英文版，将 config.js 中的文案集中管理即可。
3. **与场景二/三整合**：预留统一的色板变量、字体规范、导航栏位置，后续场景可复用相同 CSS 和 router。
4. **浏览器兼容**：目标 Chrome/Firefox/Edge 最新版，不兼容 IE。
5. **性能**：7 个 CSV 总计约 <500KB，一次性加载可接受。如数据扩大，改为按需加载。
